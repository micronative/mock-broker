Mock Broker is a simple mock message broker

## Configuration
<pre>
"require": {
        "micronative/mock-broker": "^1.0.0"
},
"repositories": [
    { "type": "vcs", "url": "https://github.com/micronative/mock-broker" }
],
</pre>

Run
<pre>
composer require micronative/mock-broker:1.0.0
</pre>
